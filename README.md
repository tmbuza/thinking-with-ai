# Thinking with AI

**A human-first guide to using AI for real work and defensible decisions**

---

## Overview

Artificial intelligence tools can generate answers quickly.

But real work requires more than answers.

It requires:
- clear thinking  
- structured reasoning  
- defensible decisions  

This guide introduces a practical framework for using AI without losing ownership of your thinking.

---

## Core Idea

> **Start with thinking. Then use AI.**

More specifically:

> **Start with a position.**

A position is your current thinking before using AI:
- a problem  
- a direction  
- or a hypothesis  

---

## What This Guide Teaches

This guide focuses on how to:

- define a clear starting position  
- use AI to extend your thinking  
- interrogate outputs critically  
- transform outputs into defensible reasoning  
- move from outputs to well-calibrated claims  
- integrate AI into real workflows  
- build a consistent personal AI workflow
